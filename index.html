Welcome to C++ Programming!

C++ is a powerful, general-purpose programming language that supports both procedural and object-oriented programming paradigms. It's widely used in system programming, game development, and high-performance applications.

Why Learn C++?

Your First C++ Program

#include <iostream>

int main() {

Variables & Data Types

Variables are containers for storing data values. In C++, you must declare the type of data a variable will hold.

Basic Data Types

  • int - Integer numbers
  • float - Floating-point numbers
  • double - Double precision floating-point
  • char - Single character
  • bool - Boolean (true/false)

Variable Declaration and Initialization

#include <iostream>
using namespace std;

int main() {
    // Variable declaration and initialization
    int age = 25;
    float height = 5.9;
    char grade = 'A';
    bool isStudent = true;
    
    // Display variables
    cout << "Age: " << age << endl;
    cout << "Height: " << height << endl;
    cout << "Grade: " << grade << endl;
    cout << "Is Student: " << isStudent << endl;
    
    return 0;
}

Control Structures

Control structures allow you to control the flow of execution in your program using conditions and loops.

Types of Control Structures

  • if-else - Conditional execution
  • for loop - Repeat code a specific number of times
  • while loop - Repeat code while condition is true
  • switch - Multiple choice selection

If-Else and Loops Example

#include <iostream>
using namespace std;

int main() {
    int number = 7;
    
    // If-else statement
    if (number > 0) {
        cout << "Positive number" << endl;
    } else if (number < 0) {
        cout << "Negative number" << endl;
    } else {
        cout << "Zero" << endl;
    }
    
    // For loop
    cout << "Counting from 1 to 5: ";
    for (int i = 1; i <= 5; i++) {
        cout << i << " ";
    }
    cout << endl;
    
    // While loop
    int count = 0;
    cout << "While loop countdown: ";
    while (count < 3) {
        cout << (3 - count) << " ";
        count++;
    }
    cout << endl;
    
    return 0;
}

Functions

Functions are blocks of code that perform specific tasks. They help organize code and make it reusable.

Function Components

  • Return Type - What the function returns
  • Function Name - Identifier for the function
  • Parameters - Input values (optional)
  • Function Body - Code that executes

Function Declaration and Usage

#include <iostream>
using namespace std;

// Function declaration
int add(int a, int b);
void greet(string name);

int main() {
    // Function calls
    int result = add(5, 3);
    cout << "Sum: " << result << endl;
    
    greet("Alice");
    
    return 0;
}

// Function definition
int add(int a, int b) {
    return a + b;
}

void greet(string name) {
    cout << "Hello, " << name << "!" << endl;
}

Arrays & Strings

Arrays store multiple values of the same type, while strings are sequences of characters.

Array and String Basics

  • Arrays - Fixed-size collections of elements
  • Strings - Character arrays or string objects
  • Indexing - Access elements using index (0-based)
  • Size - Number of elements in the array

Working with Arrays and Strings

#include <iostream>
#include <string>
using namespace std;

int main() {
    // Array declaration and initialization
    int numbers[5] = {10, 20, 30, 40, 50};
    
    cout << "Array elements: ";
    for (int i = 0; i < 5; i++) {
        cout << numbers[i] << " ";
    }
    cout << endl;
    
    // String operations
    string name = "C++ Programming";
    cout << "String: " << name << endl;
    cout << "Length: " << name.length() << endl;
    cout << "First character: " << name[0] << endl;
    
    return 0;
}

Object-Oriented Programming

Object-Oriented Programming (OOP) is a programming paradigm based on objects and classes.

OOP Concepts

  • Class - Blueprint for objects
  • Object - Instance of a class
  • Encapsulation - Data hiding
  • Inheritance - Code reuse
  • Polymorphism - Multiple forms

Class and Object Example

#include <iostream>
#include <string>
using namespace std;

class Student {
private:
    string name;
    int age;
    
public:
    // Constructor
    Student(string n, int a) {
        name = n;
        age = a;
    }
    
    // Method to display information
    void displayInfo() {
        cout << "Name: " << name << ", Age: " << age << endl;
    }
    
    // Getter method
    string getName() {
        return name;
    }
};

int main() {
    // Create object
    Student student1("Alice", 20);
    Student student2("Bob", 22);
    
    // Call methods
    student1.displayInfo();
    student2.displayInfo();
    
    return 0;
}

Call by Reference

Call by reference is a method of passing arguments to a function where the function receives a reference to the original variable instead of a copy. This allows the function to modify the original variable directly.

Key Concepts

  • Reference - An alias for an existing variable
  • Memory Efficiency - No copying of large objects
  • Direct Modification - Changes affect the original variable
  • Syntax - Use & symbol in function parameters

Basic Call by Reference

#include <iostream>
using namespace std;

// Function with reference parameter
void swap(int &a, int &b) {
    int temp = a;
    a = b;
    b = temp;
}

int main() {
    int x = 10, y = 20;
    
    cout << "Before swap: x = " << x << ", y = " << y << endl;
    
    swap(x, y);  // Call by reference
    
    cout << "After swap: x = " << x << ", y = " << y << endl;
    
    return 0;
}

Call by Value vs Call by Reference

#include <iostream>
using namespace std;

// Call by value - creates a copy
void modifyByValue(int num) {
    num = 100;
    cout << "Inside modifyByValue: " << num << endl;
}

// Call by reference - modifies original
void modifyByReference(int &num) {
    num = 100;
    cout << "Inside modifyByReference: " << num << endl;
}

int main() {
    int value = 50;
    
    cout << "Original value: " << value << endl;
    
    modifyByValue(value);
    cout << "After call by value: " << value << endl;
    
    modifyByReference(value);
    cout << "After call by reference: " << value << endl;
    
    return 0;
}

Reference with Arrays

#include <iostream>
using namespace std;

// Function to modify array elements
void modifyArray(int arr[], int size) {
    for (int i = 0; i < size; i++) {
        arr[i] *= 2;  // Double each element
    }
}

// Function to print array
void printArray(int arr[], int size) {
    cout << "Array elements: ";
    for (int i = 0; i < size; i++) {
        cout << arr[i] << " ";
    }
    cout << endl;
}

int main() {
    int numbers[] = {1, 2, 3, 4, 5};
    int size = 5;
    
    cout << "Original array:" << endl;
    printArray(numbers, size);
    
    modifyArray(numbers, size);
    
    cout << "After modification:" << endl;
    printArray(numbers, size);
    
    return 0;
}

Multiple Return Values using References

#include <iostream>
#include <cmath>
using namespace std;

// Function that returns multiple values using references
void calculateCircle(double radius, double &area, double &circumference) {
    area = 3.14159 * radius * radius;
    circumference = 2 * 3.14159 * radius;
}

// Function to find min and max of array
void findMinMax(int arr[], int size, int &min, int &max) {
    min = max = arr[0];
    for (int i = 1; i < size; i++) {
        if (arr[i] < min) min = arr[i];
        if (arr[i] > max) max = arr[i];
    }
}

int main() {
    // Example 1: Circle calculations
    double radius = 5.0;
    double area, circumference;
    
    calculateCircle(radius, area, circumference);
    cout << "Circle with radius " << radius << ":" << endl;
    cout << "Area: " << area << endl;
    cout << "Circumference: " << circumference << endl;
    
    // Example 2: Min and max of array
    int numbers[] = {45, 12, 78, 23, 56, 89, 34};
    int size = 7;
    int minVal, maxVal;
    
    findMinMax(numbers, size, minVal, maxVal);
    cout << "\nArray: ";
    for (int i = 0; i < size; i++) {
        cout << numbers[i] << " ";
    }
    cout << endl;
    cout << "Minimum: " << minVal << ", Maximum: " << maxVal << endl;
    
    return 0;
}

Important Notes

  • References must be initialized when declared
  • References cannot be reassigned to refer to another variable
  • No null references - references must always refer to a valid object
  • Memory efficient - no copying of large objects
  • Use const references when you don't need to modify the value

Const References for Read-Only Access

#include <iostream>
#include <string>
using namespace std;

// Function that doesn't modify the parameter
void printInfo(const string &name, const int &age) {
    cout << "Name: " << name << endl;
    cout << "Age: " << age << endl;
    // name = "Changed";  // This would cause compilation error
    // age = 0;           // This would cause compilation error
}

// Function to calculate string length efficiently
int getStringLength(const string &str) {
    return str.length();
}

int main() {
    string personName = "Alice";
    int personAge = 25;
    
    printInfo(personName, personAge);
    
    cout << "String length: " << getStringLength(personName) << endl;
    
    return 0;
}
